Sobel边缘算法——滤镜




边缘算法原理:

获取canvas画布中每个像素点的颜色rgb的三个值,将三个值求平均得到（灰度值）,

        var kernelX = [
                    [-1,0,1],
                    [-2,0,2],
                    [-1,0,1]];

        var kernelY =  [
                     [-1,-2,-1],
                     [0,0,0],
                     [1,2,1]];

将灰度值与KernelX,KernelY卷积因子进行梯度计算,获取X,Y两个值

通过公式[Math.sqrt((X * Y) + (X * Y))>>0]获取边缘值
